# Апи Яндекс карт

## Подключение

Описать в зависимостях.

```
{
    block : 'map',
    mods : { api : 'ymaps' },
    js : {
        center : [55.76, 37.64],
        zoom : 10 // можно не определять по умолчанию 10
    }
}
```

## Добавляем контролы на карту

```
{
    block : 'map',
    mods : { api : 'ymaps', zoomControl : true, typeSelector : true  },
    js : {
        center : [55.76, 37.64],
        zoom : 10 // можно не определять по умолчанию 10
    }
}
```
