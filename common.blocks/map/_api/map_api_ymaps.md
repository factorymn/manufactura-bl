# Апи Яндекс карт

## Подключение

Описать в зависимостях.

```
{
    block : 'map',
    mods : { api : 'ymaps' },
    js : {
        'lang': 'ru-RU', // можно не задавать
        'center': [55.76, 37.64],
        'zoom': 10, // можно не задавать
        'geoObjects': [
            {
                coords : [55.8, 37.8],
                // Можно не задвать
                properties: {
                    iconContent: '',
                    hintContent: 'Тут можно написать тайтл места'
                }
            }
        ]
    }
}
```
